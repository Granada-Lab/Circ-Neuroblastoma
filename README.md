# Circ-Neuroblastoma

Analysis code for:

> Ector C, Schmal C, Didier J, De Landtsheer S, Schulte JH, Keilholz U, Sauter T, Kramer A, Herzel H, **Granada AE**.
> *Circadian rhythm heterogeneity modulates drug response variations in neuroblastoma models.*
> **Cell Reports** 2026;45(2). DOI: [10.1016/j.celrep.2026.116975](https://doi.org/10.1016/j.celrep.2026.116975)

## Contents

Code is organised by figure panels:

| Folder | Figures |
|---|---|
| `Figure_1BCDEF_2ABC_S1_S3` | Figures 1B–F, 2A–C, S1, S3 |
| `Figure_2EFGHI` | Figures 2E–I |
| `Figure_3BCDEFG_4ABC_S5_S6AB` | Figures 3B–G, 4A–C, S5, S6A–B |
| `Figure_4EFGH_S4_S6CD` | Figures 4E–H, S4, S6C–D |

## Data

Input data files are available on Figshare:
[https://figshare.com/projects/Circadian-Rhythms-Neuroblastoma-Cells/269114](https://figshare.com/projects/Circadian-Rhythms-Neuroblastoma-Cells/269114)
